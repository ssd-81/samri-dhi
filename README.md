# Samridhi

## The Problem

**500M+ Indians save in FDs. 0 get credit for it.**

Traditional credit scores only measure debt repayment. If you've never borrowed money, you have no score — even if you've saved diligently for decades.

Your FD deposits prove discipline. But banks can't see it.

## The Solution

Samridhi measures what matters: **savings discipline**.

We analyze your Fixed Deposit history across 5 dimensions:

- **Consistency** — Do you save regularly?
- **Discipline** — Do you let FDs mature?
- **Growth** — Is your corpus growing?
- **Diversification** — Do you spread across banks?
- **Intelligence** — Do you optimize rates?

→ Generate a credit score (300-900) based on actual saving behavior.

## Why It Matters

| Before | After |
|--------|-------|
| Disciplined saver = No score | Disciplined saver = 750+ score |
| FD holdings ignored | FD holdings = Credit collateral |
| No credit products | FD-backed cards, loans, UPI credit |

## Quick Start

```bash
go build -o fd-credit-score .
./fd-credit-score
# Open http://localhost:8080
```
